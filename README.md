# Premium Insurance Category Prediction

This project predicts the insurance premium category based on user details using a Machine Learning model.
It includes:

✅ FastAPI backend (ML prediction API) 
✅ Streamlit frontend (UI app)
✅ Fully Dockerized setup (API + Frontend)
✅ Ready for cloud deployment

Step 1
**Add to your Docker Desktop from Docker Hub**
API : docker pull ankanbasak0/insurance-api:latest
Frontend : docker pull ankanbasak0/insurance-frontend:latest

Step 2
**Run on Docker Hub Terminal**
Terminal 1: docker run -p 8000:8000 ankanbasak0/insurance-api:latest
Terminal 2 : docker run -p 8501:8501 ankanbasak0/insurance-frontend:latest

Step 3
RUN on browser: localhost:8501
